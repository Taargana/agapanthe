# Protocole visuel M7 — IBL & skybox (spec §5/§6)

**Statut : à valider (revue humaine)** — captures Agapanthe produites, comparaison viewer Khronos à faire.

## Cible

Critère de sortie M7 (board) : IBL image-based complet (irradiance diffuse + prefiltered specular + BRDF LUT
+ skybox) remplaçant l'ambiant constant. Le juge de paix est **MetalRoughSpheres** : la grille varie
metallic (lignes) × roughness (colonnes), donc l'IBL doit produire la bonne signature — coin metallic+lisse =
miroir de l'environnement, metallic+rugueux = reflet flou, diélectrique = diffus mat + petit reflet Fresnel.

Environnement : `studio_small_1k.hdr` (Poly Haven, CC0), généré à 512² cubemap + irradiance 32² + prefiltered
128²×8 mips + BRDF LUT 512² en ~125 ms au chargement.

## Captures Agapanthe

- `2026-07-10-m7-metalroughspheres-agapanthe.png` — la grille, IBL + skybox.
- `2026-07-10-m7-damagedhelmet-agapanthe.png` — le casque, reflets d'environnement + ciel visible.

Cadrage et exposition par défaut (1.0 = +0.0 EV), pas de `AGAPANTHE_VIEW`.

## Procédure

1. **Agapanthe** : `DYLD_LIBRARY_PATH=/opt/homebrew/lib dotnet run --project samples/Sandbox -- MetalRoughSpheres.glb`
   - HDRI par défaut ; `AGAPANTHE_HDRI=<path.hdr>` pour en changer.
   - Cadrage/exposition par défaut. Capture headless : `AGAPANTHE_MAX_FRAMES=3 AGAPANTHE_CAPTURE=<out.ppm>`.
2. **Référence Khronos** : https://github.khronos.org/glTF-Sample-Viewer-Release/
   - Charger MetalRoughSpheres, tonemap **ACES**, IBL activé.
   - **L'environnement exact ne peut pas correspondre** (le viewer a ses propres HDRI) : juger le *caractère*
     metallic/roughness (transition miroir→flou, diélectrique mat), pas l'ambiance pixel-pour-pixel.
3. Comparer côte à côte et annoter ci-dessous.

## Critères (cocher)

| Critère | Attendu | OK ? |
|---|---|---|
| Rangée metallic — lisse | Miroir net de l'environnement (le studio se lit dans la sphère) | ☐ |
| Rangée metallic — rugueux | Reflet progressivement flou le long de la colonne roughness (mips prefiltered) | ☐ |
| Diélectrique | Diffus mat dominé par l'irradiance, petit reflet spéculaire, pas de « trou noir » | ☐ |
| Fresnel rasant | Reflet renforcé aux bords des sphères (Fresnel roughness-aware) | ☐ |
| Skybox | L'environnement remplit le fond, aligné avec les reflets ; pas de couture ni NaN | ☐ |
| Continuité IBL/direct | Pas de discontinuité dure entre zones éclairées direct et ambiant IBL | ☐ |
| Tone mapping | Hautes lumières du studio roulent doucement, pas de saturation blanche plate | ☐ |
| Pas d'artefacts | Pas de pixels NaN isolés, pas de seams de cubemap visibles | ☐ |

## Écarts attendus vs viewer Khronos

- **Environnement différent** : impossible de charger le même HDRI que le viewer sans effort ; comparer le
  caractère des matériaux, pas l'ambiance.
- **Prefiltered sur environnement single-mip** : le prefilter échantillonne le niveau de base de l'env (pas de
  pyramide de mips sur l'env) → léger surcroît de fireflies possible sur roughness élevée + HDR très contrasté.
  Raffinement M8/phase 2 si visible.
